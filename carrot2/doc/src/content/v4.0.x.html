Carrot2 4.0.x

Release history for Carrot2 4.0.x and bugfix releases.

Version 4.0.4

Release 4.0.4 contains the following changes:

  • A workaround for palantir publishing everything in POM's dependencyManagement section. This locks dependencies that should not be locked.
  • Documentation system rendered with apidocs theme.

Version 4.0.3

Release 4.0.3 contains the following changes:

  • CARROT-1276: Allow multiple classloader SPI resolution in language components loader.

Version 4.0.2

Release 4.0.2 is a republished release of broken version 4.0.1 with complete, valid artifacts.

This release contains the following changes:

  • AttrStringArray should accept Object[] with just strings inside.

Version 4.0.1

This bugfix release is broken due to only partial artifacts (a subset of JARs) uploaded to Maven Central for some reason.

Version 4.0.0

Final 4.0.0 release of Carrot2, with the new API.
